== AsciiDoc 支持

本示例工程完整支持 AsciiDoc 语法，包括：

=== 文本格式化

*粗体*、_斜体_、`等宽字体`、*_*粗斜体*_

=== 列表

==== 无序列表
* 第一项
* 第二项
** 子项 2.1
** 子项 2.2
* 第三项

==== 有序列表
. 第一步
. 第二步
. 第三步

==== 描述列表
术语一:: 术语一的解释
术语二:: 术语二的解释

=== 代码块

[source,javascript]
----
function greet(name) {

greet('AsciiDoc');

== Docusaurus 特性

=== 版本管理

Docusaurus 支持文档版本管理，适合 API 文档或多版本文档场景。

=== 国际化

支持多语言配置，可以为不同语言提供独立的文档内容。

=== 搜索

内置 Algolia DocSearch 集成，提供全文搜索能力。

=== 自定义主题

可以通过 React 组件自定义页面布局和样式。
