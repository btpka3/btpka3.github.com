=== 带行号的代码

[source,java,linenums]
----
package com.example;

import java.util.List;
import java.util.stream.Collectors;

public class StreamExample {
    public static void main(String[] args) {
        List<String> names = List.of("Alice", "Bob", "Charlie", "David");

        List<String> filtered = names.stream()
            .filter(name -> name.length() > 3)
            .map(String::toUpperCase)
            .collect(Collectors.toList());

        filtered.forEach(System.out::println);
    }
}
----

=== 多种语言示例

[source,python]
----
def fibonacci(n: int) -> list[int]:
    """生成斐波那契数列"""
    fib = [0, 1]
    for i in range(2, n):
        fib.append(fib[i-1] + fib[i-2])
    return fib[:n]

print(fibonacci(10))
----

[source,go]
----
package main

import "fmt"

func fibonacci(n int) []int {
    fib := make([]int, n)
    fib[0], fib[1] = 0, 1
    for i := 2; i < n; i++ {
        fib[i] = fib[i-1] + fib[i-2]
    }
    return fib
}

func main() {
    fmt.Println(fibonacci(10))
}
----

[source,rust]
----
fn fibonacci(n: usize) -> Vec<u64> {
    let mut fib = vec![0, 1];
    for i in 2..n {
        let next = fib[i - 1] + fib[i - 2];
        fib.push(next);
    }
    fib.truncate(n);
    fib
}

fn main() {
    println!("{:?}", fibonacci(10));
}
----

== 警告块 (Admonitions)

[NOTE]
====
这是一个**提示**信息块。用于提供补充说明或背景信息。

支持包含多段内容和格式化。
====

[TIP]
====
使用 `npm run clear` 可以清除 Docusaurus 缓存，解决许多构建问题。
====

[IMPORTANT]
====
在 `package.json` 中必须包含 `overrides` 字段来锁定 webpack 版本，否则可能出现构建失败。
====

[WARNING]
====
不要在生产环境直接使用 `npm start`，它仅用于开发。生产环境应该使用 `npm run build` 后通过静态服务器部署。
====

[CAUTION]
====
修改 `docusaurus.config.mjs` 后需要重启开发服务器才能生效。热更新不会自动加载配置变更。
====

== Include 指令（概念展示）

AsciiDoc 支持通过 `include` 指令引入外部文件内容：

[source,asciidoc]
----
\include::shared/header.adoc[]

\include::examples/code-sample.java[tags=main]

\include::data/config.yaml[lines=5..15]
----

这使得大型文档可以模块化管理，支持按标签和行号精确引入。

AsciiDoc 是由 Stuart Rackham 于 2002 年创建的标记语言。footnote:[最初设计目标是成为 DocBook XML 的轻量级替代方案。]

Docusaurus 由 Meta (前 Facebook) 开发维护。footnote:[最初用于 React Native 和 Jest 的文档站点。]

== 文字替换与符号

AsciiDoc 支持自动文字替换：

* (C) 变为版权符号
* (R) 变为注册商标
* (TM) 变为商标
* -> 变为箭头
* => 变为双箭头
* -- 变为破折号

== 条件内容（概念展示）

AsciiDoc 支持条件包含/排除内容：

[source,asciidoc]
----
\ifdef::env-github[]
此内容仅在 GitHub 渲染时显示。
\endif::[]

\ifndef::env-github[]
此内容在非 GitHub 环境中显示。
\endif::[]

\ifeval::["{backend}" == "html5"]
此内容仅在 HTML5 输出时包含。
\endif::[]
----
