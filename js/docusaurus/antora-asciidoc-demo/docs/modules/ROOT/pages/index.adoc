= Antora AsciiDoc 示例站点

欢迎来到 Antora + AsciiDoc + antora-ui-spring 示例站点。

== 功能特性

* 浏览器前端本地搜索（Pagefind）
* 国际化支持：中文（默认）、英文
* 多版本文档管理
* 代码高亮 + 行号
* 丰富的 AsciiDoc 语法示例
* 宽屏显示优化

== 本地启动

[source,bash]
----
# 安装依赖
npm install

# 构建站点 + 生成搜索索引
npx antora antora-playbook.yml --to-dir=build/site
npx pagefind --site build/site

# 启动本地服务（访问 http://localhost:8888）
npx http-server build/site -p 8888
----
