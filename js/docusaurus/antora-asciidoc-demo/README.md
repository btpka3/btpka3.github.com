

该工程是 antora + asciidoc + link:https://github.com/spring-io/antora-ui-spring[antora-ui-spring] 生成静态文档站点的示例。

需要包含以下内容

* 浏览器前端 local search
* i18n : 中文（默认），英文（可以是简单示例说明）
* 多版本 。默认版本需要详细，其他版本仅简单示例说明即可。
* 常见的代码块儿要能多支持多语言着色（java,shell,javascript等），最好能支持行号等高级特性。
* 给出一个尽可能详细的 asciidoc 示例。比如包含列表，表格， PlantUML , mermaid, kbd 等。
* 对宽屏 比如（2560 x 1440）能显示优化，使得空白区域减少。
* 给出一个从零构建上述完整功能的说明文档。
* 提示：spring 相关的文档网站有使用 antora-ui-spring
